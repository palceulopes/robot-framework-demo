🔑 CLASSES PRINCIPAIS

┌─ AdbMock
│  └─ Simula Android Debug Bridge
│     • get_property(name) - Retorna propriedade
│     • shell_command(cmd) - Executa comando
│     • set_property(name, value) - Define propriedade
│
├─ CanBusManager
│  └─ Gerencia barramento CAN virtual
│     • send_signal(msg, sig, value) - Envia sinal
│     • inject_speed_signal(kmh) - Injeta velocidade
│     • get_message_names() - Lista mensagens
│     • get_message_signals(msg) - Lista sinais
│     • close() - Fecha CAN
│
└─ AutomotiveLibrary
   └─ Integração com Robot Framework
      • get_device_property(prop) - Propriedade ADB
      • inject_speed_kmh(speed) - Injeta velocidade
      • get_can_messages() - Mensagens CAN
      • get_can_signals(msg) - Sinais CAN
